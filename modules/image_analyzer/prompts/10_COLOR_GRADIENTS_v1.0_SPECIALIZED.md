# COLOR AND GRADIENTS ANALYSIS PROMPT
## Vision API Specialized Template v1.0

### ANALYSIS OBJECTIVE
Analyze color charts, gradient scales, calibration patterns, and color-based technical diagrams with focus on color accuracy, gradient transitions, and calibration measurements.

---

## ANALYSIS METHODOLOGY

### PHASE 1: COLOR PATTERN IDENTIFICATION
**Identify the primary color analysis type:**
- **Color Charts**: Discrete color swatches or palettes
- **Gradient Scales**: Continuous color transitions
- **Calibration Patterns**: Test patterns for display validation
- **Color Coding Systems**: Color-coded technical information
- **Mixed Color Diagrams**: Combination of above types

### PHASE 2: COLOR SPACE ANALYSIS
**Analyze color characteristics:**
- **Primary Colors**: Identify RGB, CMYK, or other color space usage
- **Color Range**: Document color spectrum coverage
- **Saturation Levels**: Analyze color intensity variations
- **Brightness Levels**: Document luminance variations
- **Color Temperature**: Identify warm/cool color characteristics

### PHASE 3: GRADIENT AND TRANSITION ANALYSIS
**For gradient patterns:**
- **Gradient Direction**: Horizontal, vertical, radial, or complex
- **Transition Smoothness**: Smooth, stepped, or discrete transitions
- **Color Stops**: Identify key color points in gradients
- **Gradient Type**: Linear, radial, angular, or complex patterns
- **Transition Quality**: Assess smoothness and continuity

### PHASE 4: CALIBRATION AND MEASUREMENT ANALYSIS
**For calibration patterns:**
- **Test Pattern Type**: Color bars, checkerboards, gradients, or custom
- **Measurement Scales**: Identify any numerical or percentage scales
- **Reference Standards**: Note any standard color references (sRGB, Adobe RGB, etc.)
- **Calibration Purpose**: Display testing, printer calibration, camera calibration
- **Quality Indicators**: Identify any pass/fail or quality metrics

### PHASE 5: TECHNICAL COLOR INFORMATION EXTRACTION
**Extract technical specifications:**
- **Color Values**: RGB, HEX, CMYK, or other color specifications
- **Color Names**: Standard color names or custom designations
- **Color Relationships**: Color harmony, contrast, or matching information
- **Usage Instructions**: How colors should be applied or interpreted
- **Standards Compliance**: Reference to color standards or specifications

## SPECIALIZED ANALYSIS TECHNIQUES

### COLOR ACCURACY ASSESSMENT
- Compare colors against known standards when references are visible
- Identify color shifts, casts, or reproduction issues
- Note any color gamut limitations or clipping

### GRADIENT QUALITY EVALUATION
- Assess smoothness of color transitions
- Identify banding, posterization, or discontinuities
- Evaluate gradient uniformity and consistency

### CALIBRATION PATTERN ANALYSIS
- Identify test pattern purpose and methodology
- Extract measurement scales and reference values
- Assess pattern completeness and quality

### AUTOMOTIVE COLOR STANDARDS
- Reference ISO 3864 for safety colors
- Consider SAE standards for automotive displays
- Evaluate visibility under various lighting conditions

## CRITICAL ANALYSIS POINTS

### TECHNICAL ACCURACY
- Verify color value specifications against visible colors
- Check consistency between different color representations
- Validate measurement scales and calibration references

### FUNCTIONAL ASSESSMENT
- Evaluate color choices for intended application
- Assess visibility and readability under various conditions
- Consider color accessibility and universal design principles

### QUALITY VALIDATION
- Identify any color reproduction issues or artifacts
- Assess pattern quality and measurement precision
- Evaluate compliance with relevant standards
