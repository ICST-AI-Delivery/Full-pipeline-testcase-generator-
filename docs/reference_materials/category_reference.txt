AUTOMOTIVE DIAGRAM CLASSIFICATION CATEGORIES
=====================================================

Use these 11 categories when filling in the Ground_Truth_Category column:

1. TELLTALE ICONS & INDICATORS
   - Automotive telltale symbols
   - Dashboard warning lights and status indicators
   - Individual icon symbols

2. CONFIGURATION TABLES
   - Technical data tables with rows and columns containing text, numbers, codes, and parameters
   - May include colored cells, highlighted headers, table formatting, and visual organization elements like borders or shading
   - Focus is on structured data presentation, NOT automotive warning symbols
   - Examples: parameter tables, specification sheets, configuration matrices with colored formatting

3. HMI DISPLAY LAYOUTS
   - Complete instrument cluster display interfaces
   - Dashboard layouts and gauge clusters
   - UI wireframes and screen designs
   - Full display mockups

4. STATE EVENT MATRICES
   - State matrix diagrams with grid patterns
   - State transitions and input/output relationships
   - Matrix-style logic diagrams

5. MULTI-CONDITION LOGIC TABLES
   - Complex tables with AND/OR conditional relationships
   - Decision logic tables
   - Multi-variable condition matrices

6. SYSTEM ARCHITECTURE DIAGRAMS
   - System architecture showing components and connections
   - Data flow and technical system design
   - Block diagrams with interconnections

7. PROCESS FLOW DIAGRAMS
   - Workflow or process diagrams
   - Flowchart symbols showing sequential processes
   - Decision trees and process flows

8. TECHNICAL SPECIFICATIONS
   - Technical parameter documentation
   - Performance characteristics and specifications
   - Detailed technical requirements

9. WIRING & SIGNAL DIAGRAMS
   - Electrical connections and signal routing
   - Wiring documentation
   - Circuit diagrams

10. TIMING DIAGRAMS
    - Signal timing sequences and waveforms
    - Protocol timing diagrams
    - Time-based signal representations

11. TABLE + TELLTALES
    - Tables containing actual automotive telltale icons, warning symbols, or dashboard indicators embedded within table cells
    - Must contain recognizable automotive symbols (warning lights, status icons, dashboard symbols) as cell content
    - The telltale symbols are the primary visual content, not just table organization elements
    - NOT just colored formatting or text highlighting - requires actual automotive warning icons

12. others

13. color details??
 - color gradients 
INSTRUCTIONS:
- Choose the category that best describes the primary content of each image
- If an image contains multiple elements, choose the dominant/primary element
- Use exact category names as listed above (copy/paste recommended)
- Add notes in the Notes column for any special cases or uncertainties
