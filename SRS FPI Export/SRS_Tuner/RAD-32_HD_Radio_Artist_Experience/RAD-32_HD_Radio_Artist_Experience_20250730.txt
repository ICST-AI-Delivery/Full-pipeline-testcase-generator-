5. ID: 4097857
- Contents: [Info] The Artist Experience bandwidth is pooled across all HD channels, so the images for the MPS/HD1,
SPS1/HD2 and other channels share the same data bandwidth
[Info]The data size allocated for transmission of Artist Experience is 4 kbps. So, a maximum file size
image of 24 kilobytes would take 48 seconds to transmit
[Info]iBiquity Recommended Storage Capacity Example for Artist Experience®:
Non-Volatile Memory (Permanent) – Station Logos
. Maximum logo size = 24 kbytes x 50 logos = 1.17 Mbytes (as defined by RD_HD-05)
.Volatile Memory (RAM) – Album Art
.Maximum image size = 24 kbytes x 2 Album Art images per channel x 4 (SPS Channels) x 2 frequencies (to save last station for return tuning) = 384 kbytes (as defined by RD_HD-05.
- Artifact Type: Information
- State (Harman_Requirement): New
