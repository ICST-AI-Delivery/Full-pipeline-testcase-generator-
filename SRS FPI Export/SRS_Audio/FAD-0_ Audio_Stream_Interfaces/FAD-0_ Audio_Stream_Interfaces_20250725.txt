### Primary Requirements:

1. ID: 3524078
- Contents: The system shall have the stream activation interface to detect the presence of an active audio source in the stream.
- Artifact Type: SRS Req
- Responsible Domain: Audio Control
- Test Stage: System Qualification Test
- Verification Criteria: Play an audio source.

Verify  by using the corresponding CAN signal, the stream activation interface has the following values:
1 ("Active"): when an audio source is active on the stream.

0 ("Not_Active"): when an audio source is not active on the channel.

Verify the value of the CAN signal(e.g. IDC_AudioStreams.EntertainmentStreamActivation for Entertainment Audio source).

Note: Refer to the artifact 3731533 (IDC-418040) for stream activation interfaces CAN signal.
