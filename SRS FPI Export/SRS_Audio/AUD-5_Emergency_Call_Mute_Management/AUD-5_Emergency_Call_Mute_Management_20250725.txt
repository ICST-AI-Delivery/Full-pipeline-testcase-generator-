### Primary Requirements:

1. ID: 3494603
- Contents: If proxi parameter "EmergencyCallMute" is enabled, the system shall enable the emergency call mute feature.
- Artifact Type: SRS Req
- Responsible Domain: IOC_IOC
- Test Stage: System Qualification Test
- Verification Criteria: Flash a SW which has proxi parameter EmergencyCallMute = 1 

Verify via log the emergency call mute feature is enabled.

The emergency call mute feature should be disabled if proxi parameter EmergencyCallMute = 0
- Grooming Status: Approved

- Artifact Type: SRS Req
- Responsible Domain: IOC_IOC
- Test Stage: System Qualification Test
- Verification Criteria: Precondition: Emergency call mute feature is enabled.

Set the hardwired signal IDCMuteReq = "Active"

Verify that the external Amplifiers are muted.
Verify the CAN signal HMI_INFO.Ecall_IDCMuteReq = 1 ("Active")

- Expert Domains: Audio Control, Kernel
- RQA Score: 90
- Issues Found by RQA: Passive voice
