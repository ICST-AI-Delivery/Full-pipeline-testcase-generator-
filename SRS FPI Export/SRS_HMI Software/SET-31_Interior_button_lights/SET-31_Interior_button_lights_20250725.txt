- Grooming Comments: Response from Ferrari: 

In the case where the RDI parameter Day_Brightness_Adj = Not_Fixed, IDC applies the same logic for Day and Night for the brightness level of interior buttons.

In the other case, where the RDI parameter Day_Brightness_Adj  = Fixed, then based on the value of input signal NightDaySts, IDC shall consider a different logic for Day (fixed value of interior button brightness) and for Night (depending on the level selected by the user) as defined in SET-31 in IDC Finalized specification and in Vehicle function SFS F041.
