2. ID: 4613203
- Contents: The system shall visualize the below elements in Classic view of Parking system with Surround view:
Bird view Area: it shows front and rear obstacles detection around the car model
Views stream frame Area: it shows the rear/front/left/right/topfront/toprear camera view.
Parking lines lines and maximum turning lines: they are shown inside the Views Stream
Frame Area and the bird view Area                                                                      Flagging lines: they are shown inside the bird view area
Info/popup area: it is positioned inside the Views Stream Frame Area, it shows the
system information (e.g Rear Camera label, calibration failure, distance from closest
obstacle, RCTA). It shall not cover the homologative area of the stream view.
Controls: front parking sensors button and camera view buttons ( positioned inside the
Bird view Area) and wide view bar (it is positioned inside Views stream Frame Area)
- Artifact Type: Information
- Responsible Domain: Parking
- Test Stage: System Qualification Test
- Verification Criteria: Precondition:
Parking system with Surround view is activated

In 2D view, activate Classic View.
- Grooming Status: Approved
- Expert Domains: Android HMI
- RQA Score: 100
- Issues Found by RQA: No issues found by RQA
- State (Harman_Requirement): Approved
