5. ID: 3845471
- Contents: Since information that have to be transmitted from NSP to IDC require more than 8 bytes, the CAN message
STATUS_NSP2 is Multimessage. This means that NSP node divides the information in more messages and, in each of them,
indicates the total number of messages and a message counter though the following two messages:
TotalFrameNumber
0 = Not used
1 = Two frames
FrameNumber, that indicate the number of the frame
0 = frame 1
1 = frame 2
The first frame is relative to rear sensor while the second is relative to frontal ones. If PROXI parameter Parking Sensor = 1
(Front&Rear) the two frames will be used by the HMI to visualize obstacles, instead, if PROXI parameter Parking Sensor = 0
(Rear), the second frame will be ignored by the user ECU. [ Approved ]
- Artifact Type: Information
- Responsible Domain: Parking
- State (Harman_Requirement): New
