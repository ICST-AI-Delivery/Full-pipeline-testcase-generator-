- Contents: System shall perform check phase at key on event for a time duration of T_Diagnosis
Note : T_Diagnosis = T_Lampcheck + 1000ms
- Artifact Type: SRS Req
- Responsible Domain: Cluster SW
- Test Stage: System Qualification Test
- Verification Criteria: Verify bub check for duaration T_Lampcheck (4s).
Verify other errors not reported until T_Diagnosis finished
- Grooming Status: Approved
- Expert Domains: Cluster SW, System Infra
- RQA Score: 80
- Issues Found by RQA: Missing unit 
 (Look for: "‘1000’ has no units and has no tolerance. Add a unit, and, unless this refers to countable items, it should have a tolerance, such as less than, or +/- 5%" )
- State (Harman_Requirement): Approved
- Requirement Rationale: Note: after the “T_Lampcheck” time parameter, IDC shall wait 1000 ms before signalling other possible failures (except for the ones that can interrupt this cycle and that
shall be signaled when incoming), in order to guarantee that all the requests of telltale-check handled by external ECUs are terminated, within the “T_Lampcheck” time parameter.
