2. ID: 5106249
- Contents: IDC shall set STATUS_NIT_V2.AQS_Conditions according to information retrieved by ADASIS as follows:
if RoadInResidentalArea == true -> AQS_Conditions = 1 ("city_mode)
if  RoadInTunnel == true or "FormOfWay = 11: Poor condition road" or "FormOfWay = 12: Unpaved road" classification -> AQS_Conditions = 3 ("tunnel_mode")
if "FormOfWay == 5: Major slip road" in conjunction with the "FormOfWay = 1: Motorway" -> AQS_Conditions = 2 ("highway_mode")
if not city_mode or one of the other modes available -> AQS_Conditions = 4 ("country_mode")
for cases not covered above -> AQS_Conditions = 7 ("Invalid")
- Artifact Type: SRS Req
- Responsible Domain: ADAS_Horizon
- Test Stage: System Integration Test, Third Party Test
- Verification Criteria: verify system able to set STATUS_NIT_V2.AQS_Conditions in different conditions as mentioned
